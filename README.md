# The-Homework-Machine
This model contains the use of a Qwen-3b-instruct model to answer question along with PIL and reportlab for pdf manipulation to create high quality question answer pdfs.

### How to use
Use the colab file alogng with the font and image and run all. This allows the text written to be converted to pdf. It also contains speech to pdf(optional).


### Check quality
Please see the output.pdf once to see if it is to your likings.
